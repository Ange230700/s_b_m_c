# Features

### Ready-to-wear Module (TOP PRIORITY)

- Online catalog showcasing ready-to-wear collections
- E-commerce integration for direct sales
- Size chart and fitting guide management
- Payment integration _(e.g., Orange Money, MTN Mobile Money, Moov Money, Wave, Stripe)_

### Client Intake System

- Client file + fabrics received
- Vision/model notes
- Model Sketches upload
- Client validation tracking

### Measurement Form

- Shareable, Searchable measurement forms
- Connects to QC checklist (Optional)

### Dual Production Calendar

- Atelier calendar (internal production schedule)
- Client calendar (appointments shared with clients)
- Automated reminders for appointments and delivery dates
- Dashboard overview of upcoming tasks and deadlines

### Production pipeline

- Status tracking: Not Started → Cutting → Sewing → QC → Ready
- Assign tasks to team members
- Deadlines per step

### Quality Control

- Mandatory QC checklist before delivery
- Comparison with original measurements
- Flag errors
- Photo upload for reference

### Inventory Management

- Track fabrics, materials, and supplies
- Low stock alerts
- Supplier information

### Performance and payroll

- Productivity indicators
- Task history
- Bonus system
